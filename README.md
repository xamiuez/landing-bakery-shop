# 🥐 Landing Page - Bakery Shop

Учебный лендинг для пекарни. Первый проект - фокус на освоении базовой HTML-структуры, БЭМ-методологии и SCSS

---

## 🚀 Функционал

- Шапка сайта с логотипом и навигационным меню
- Hero-секция с фоновым изображением и карточкой спецзаказа
- Секция описания с иконками и текстовым блоком
- Секция заказа с встроенной Google Maps
- Футер с повторной навигацией
- Базовая БЭМ-структура классов

---

## 🛠 Стек технологий

- HTML5
- CSS3 / SCSS
- Методология: BEM
- Flexbox
- Git & GitHub

## 📌 Планы по улучшению

**Адаптивность**

- Реализовать адаптивную вёрстку для трёх брейкпоинтов: mobile, tablet, desktop
- Убрать все фиксированные значения размеров и заменить на резиновые значения или ограничители
- Переработать позиционирование карточки на hero (`left: 45.28%`, `top: 60.93%`) - сделать управляемым на всех экранах

**Стили и SCSS**

- Разбить монолитный `style.scss` на отдельные файлы
- Вынести все цвета и размеры в SCSS-переменные (`$color-primary`, `$font-size-base` и т.д.) и поменять формат на `hsl`
- Заменить все плавающие и магические числа на переменные или `clamp()`

**Шрифты**

- Конвертировать шрифты из `.ttf` в `.woff2` `

**HTML и семантика**

- Добавить `width` и `height` атрибуты всем `<img>` - для фикса CLS
- Добавить осмысленные `alt` у иконок в секции описания
- Добавить `loading="lazy"` атрибут к картинкам

**Производительность и качество**

- Оптимизировать фоновое изображение: конвертировать в `.webp`, добавить сжатие
- Убрать дублирующиеся CSS файлы в `/src/src/css/`

**Функционал**

- Добавить бургер-меню для мобильной версии
- Добавить плавную прокрутку и hover-эффекты на ссылки
